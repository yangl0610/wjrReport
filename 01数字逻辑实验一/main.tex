\section{预习报告（10分）}
\subsection{实验综述}

本实验要求设计一个 8421BCD 码至七段数码管显示译码电路，在 FPGA 开发板上利用四个拨码开关输入一位十进制数字（0$\sim$9）的 8421BCD 编码，经组合逻辑译码后驱动七段数码管显示相应的十进制数字。

实验采用 EP1C3 FPGA 开发板，通过 Quartus II 软件进行原理图设计，输入为四个拨码开关（对应 8421BCD 的 D、C、B、A 四位），输出为七段数码管的 a$\sim$g 七段控制信号（共阳极，低电平有效）。

\subsection{实验原理要点}

\subsubsection{FPGA 简介}
FPGA（Field Programmable Gate Array，现场可编程门阵列）是一种硬件可重构的集成电路芯片。通过硬件描述语言（VHDL 或原理图）编写数字逻辑电路，FPGA 将程序映射为实际的门级硬件电路并执行。其基本结构包括逻辑单元（LE）、嵌入式 RAM、乘法器等资源，可实现组合逻辑、时序逻辑、寄存器等各类数字电路，且可无限次重新编程。

\subsubsection{拨码开关输入}
四位拨码开关作为 8421BCD 码输入信号。拨至 ON 位置时输入为低电平（0），拨至 OFF 位置时输入为高电平（1）。实验选用右排拨码开关的 3、4、5、6 位，对应引脚为 P50、P51、P53、P54（实验板管脚对应关系见附件）。

\subsubsection{各段输出表达式推导}

对有效输入（DCBA = 0000$\sim$1001），利用卡诺图法化简各段输出（0 表示该段点亮，1 表示熄灭）。无效 BCD 输入（$D=1, C=0, A=1$ 等组合）统一令各段为 1，使显示熄灭。

\bigskip
\textbf{段 a：}输出为 0 的输入：\{0, 2, 3, 5, 6, 7, 8, 9\}，化简得
\[
\boxed{a' = D + C + BA + B'A'},\qquad a = \overline{D + C + BA + B'A'} = \overline{D}\,\overline{C}\,\overline{BA}\,\overline{B'A'}
\]

\textbf{段 b：}输出为 0 的输入：\{0, 1, 2, 3, 4, 7\}，化简得
\[
\boxed{b' = B'},\qquad b = B
\]

\textbf{段 c：}输出为 0 的输入：\{0, 1, 2, 6, 8\}，化简得
\[
\boxed{c' = C},\qquad c = \overline{C}
\]

\textbf{段 d：}输出为 0 的输入：\{0, 1, 3, 4, 5, 6, 7, 8, 9\}，化简得
\[
\boxed{d' = D'C'B'A' + C'B'},\qquad d = \overline{D'C'B'A' + C'B'}
\]

\textbf{段 e：}输出为 0 的输入：\{0, 2, 6, 8\}，化简得
\[
\boxed{e' = D'C'A + D'B'A'},\qquad e = \overline{D'C'A + D'B'A'}
\]

\textbf{段 f：}输出为 0 的输入：\{0, 4, 5, 6, 7, 8, 9\}，化简得
\[
\boxed{f' = D'C'A + D'B'A' + D'C'},\qquad f = \overline{D'C'A + D'B'A' + D'C'}
\]

\textbf{段 g：}输出为 0 的输入：\{2, 3, 4, 5, 6, 9\}，化简得
\[
\boxed{g = D + C + B + A}
\]

\subsection{实验重点（3分）}
\begin{enumerate}
  \item 理解 8421BCD 码的编码规则，掌握七段数码管的显示原理；
  \item 学会使用卡诺图法对组合逻辑电路进行化简；
  \item 掌握 Quartus II 原理图输入法设计组合逻辑电路的基本流程。
\end{enumerate}

\subsection{实验难点（2分）}
\begin{enumerate}
  \item 组合逻辑表达式的卡诺图化简，特别是多变量时的最简与或式推导；
  \item FPGA 原理图设计中的元件选用、连线和管脚分配；
  \item 时序仿真波形的正确设置与结果分析。
\end{enumerate}

\section{结果与分析（60分）}
\subsection{电路设计（20分）}

根据上述布尔表达式，在 Quartus II 中采用原理图输入法实现七段译码器。设计中使用的逻辑门元件均从 Primitives/logic 库中调用，包括：
\begin{itemize}
  \item \textbf{与门（and）}：多输入与门，实现乘积项
  \item \textbf{或门（or）}：多输入或门，实现求和项
  \item \textbf{非门（not）}：单输入非门，实现取反
\end{itemize}

各段输出 $a\sim g$ 分别由对应的布尔表达式控制，输出至七段数码管。

\end{enumerate}
